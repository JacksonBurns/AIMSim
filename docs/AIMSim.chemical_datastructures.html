Bases: object

An abstraction of a molecule

Attributes:
mol_graph (RDKIT mol object): Graph-level information of molecule.

Implemented as an RDKIT mol object.

mol_text (str): Text identifier of the molecule. mol_property_val (float): Some property associated with the molecule.

This is typically the response being studied. E.g. Boiling point, Selectivity etc.

descriptor (Descriptor object): Vector representation of a molecule.

Commonly a fingerprint.

match_fingerprint_from(reference_mol)

If target_mol.descriptor is a fingerprint, this method will try to calculate the fingerprint of the self molecules. If this fails because of the absence of mol_graph attribute in target_molecule, a ValueError is raised.

Args:

reference_mol (AIMSim.ops Molecule): Target molecule. Fingerprint of this molecule is used as the reference.
