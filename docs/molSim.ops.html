molSim.ops.similarity_measures module¶

This module contains methods to find similarities between molecules. Notes —– Input to all methods is an rdkit fingerprint or numpy array.

Supported descriptor datatypes¶

‘numpy’: np.array ‘rdkit’: rdkit.DataStructs.cDataStructs.ExplicitBitVect

Supported Metrics¶

‘tanimoto’: Jaccard Coefficient/ Tanimoto Similarity

0 (not similar at all) to 1 (identical)

‘neg_l0’: Negative L0 norm of |x1 - x2| ‘neg_l1’: Negative L1 norm of |x1 - x2| ‘neg_l2’: Negative L2 norm of |x1 - x2|
